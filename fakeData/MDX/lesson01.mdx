<h2 id="02">วิธีการเข้าถึง (access) ข้อมูลใน List โดยใช้ Index</h2>

-   ใส่เลข Index ไว้ใน bracket (`[]`) หลังชื่อตัวแปร List เพื่อเข้าถึงข้อมูลใน List ณ Index ที่ต้องการ

<SyntaxHighlight language="python">
{`scores = [60, 70, 85, 66, 70, 0, 15, 9]

<h2 id="03">วนลูปข้อมูลใน List โดยใช้ index</h2>

<SyntaxHighlight language="python">
    {`days = ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday']
index = 0
while index < 6:
    day = days[index]
    print(index, "days[%d]: %s" % (index, day))
    index += 1`}
</SyntaxHighlight>
